3.4 introduces the median and mode for random variables along with data sets and the failure rate function.

Median

A number \( m \) is considered to be the median of a random variable \( X \) if the probability that \( X \) is less than or equal to \( m \) is at least 0.5, and the probability that \( X \) is greater than or equal to \( m \) is at least 0.5:

$$P(X\le m)\ge\frac{1}{2}$$

$$P(X\ge m)\ge\frac{1}{2}$$

For example, we roll a die: the probability of rolling any specific number in the range \( \{1,2,3,4,5,6\} \) is exactly 1/6. If we test the value three for instance:

$$P(X\le 3)=P(1)+P(2)+P(3)=\frac{3}{6}=\frac{1}{2}$$

$$P(X\ge 3)=P(3)+P(4)+P(5)+P(6)=\frac{4}{6}=\frac{2}{3}\ge\frac{1}{2}$$

Since both conditions are met, three is a median. But, if we test the value four as well:

$$P(X\le 4)=P(1)+P(2)+P(3)+P(4)=\frac{4}{6}=\frac{2}{3}\ge\frac{1}{2}$$

$$P(X\ge 4)=P(4)+P(5)+P(6)=\frac{3}{6}=\frac{1}{2}$$

As we can see, four is also a median. For this scenario, any number in the interval \( [3,4] \) satisfies the conditions for being a median. Medians for random variables are not always unique, and there can be more than one.

If \( X \) is a continuous random variable with a pdf \( f \) that is positive on an interval, the median \( m \) is the value where the cdf equals exactly 0.5​:

$$F(m)=\frac{1}{2}$$

Essentially, the median is the middle point where the total probability (which is 1) is split into two equal halves of 0.5.

Mode

A number \( x_{m} \)​ is considered a mode of a random variable \( X \) (which has a pmf or pdf \( f \)) if:

$$f(x_{m})\ge f(x)$$

The mode represents the global maximum of the pdf/pmf. If a distribution is multimodal, it contains multiple local maxima, which are referred to as local modes.

Example 1

Let's try to find the median and mode of the standard normal distribution. If you recall from 3.2 its pdf is defined as:

$$f(x)=\frac{1}{\sqrt{2\pi}}e^{-\frac{1}{2}x^{2}}$$

By our definition, the median \( m \) must satisfy \( F(m)=0.5 \) (cdf). Since the standard normal distribution is perfectly symmetric, exactly half of the area under the curve lies to the left of zero, and half lies to the right, therefore the median is zero.

And as for the mode, it is the value \( x_{m} \)​ where the pdf reaches its global maximum. For the standard normal distribution, the pdf reaches its highest point when the exponent is zero. This occurs when \( x=0 \). So again, the mode is zero.

Example 2

Let's find the median and mode for the exponential distribution also from 3.2 now. First the median, remember the cdf from the exponential distribution is:

$$F(x)=1-e^{-\lambda x}$$

Setting \( F(m)=0.5 \):

$$1−e^{−\lambda m}=0.5$$

$$0.5=e^{−\lambda m}$$

Take the natural log to get rid of \( e \):

$$\text{ln}(0.5)=-\lambda m$$

$$-\text{ln}(2)=-\lambda m$$

Finally solve for \( m \):

$$m=\frac{\text{ln}(2)}{\lambda}$$

And this result is our median!

Again, the mode is the value \( x_{m} \)​ that maximizes the pdf. The pdf for the exponential distribution is:

$$f(x)=\lambda e^{-\lambda x}$$

This is a strictly decreasing function for \( x\ge0 \). Therefore, the maximum value occurs at the boundary, \( x=0 \), and that is our mode.

Data Sets

Imagine we have the finite set \( D=\{x_{1}​,x_{2}​,\cdots,x_{n}\} \). We can classify this as, well, a set of data. And we can find the mean of it by finding the average of all the values in the set. Like so:

$$\frac{1}{n}\sum_{k=1}^{n}x_{k}$$

The median of a data set is the middle value. To find it, we must first arrange the data in ascending order: \( x_{1}​\le x_{2}​\le\cdots\le x_{n}​ \). The calculation then depends on whether the number of data points (\( n \)) is even or odd. If even:

$$\frac{1}{2}\left( x_{\frac{n}{2}}+x_{\frac{n}{2}+1} \right)$$

If odd:

$$x_{\frac{n+1}{2}}$$

The mode is simply defined as the most frequent value/values within the set.

Failure Rate Function

The failure rate function is typically used for random variables that measure the lifetimes of items or objects. Let \( T \) be a random variable representing lifetime with a pdf \( f \) and a cdf \( F \). The failure rate function \( r(t) \) is then defined as:

$$r(t)=\frac{f(t)}{1-F(t)}$$

  • \( F(t)=P(T\le t) \) - this is the probability that the object fails at or before time \( t \)
  • \( 1−F(t)=P(T\gt t) \) - this represents the probability that the object survives beyond time \( t \)

Essentially \( r(t) \) provides a measure of the failure risk for objects that have already survived up to a certain time \( t \).

Survival Function

The survival function \( G(t) \) is defined as the probability that an object continues to survive beyond time \( t \):

$$G(t)=1-F(t)=P(X\gt t)$$

There is a theorem that links the survival function \( G(t) \) to the failure rate function \( r(t) \):

$$G(t)=e^{-\int_{0}^{t}r(y)dy}$$

Example

Let's assume a certain insect lives for a median age of two months, and its death rate is \( r(t)=at^{2} \) for \( t\ge0 \). First we can try to solve for the constant \( a \). To do this we can use the definition of the median. The median \( m=2 \) is the point where the probability of surviving is 0.5: \( G(2)=0.5 \). Knowing what we just learned from above, we can setup:

$$G(2)=e^{-\int_{0}^{2}ay^{2}dy}=0.5$$

After integrating we get:

$$e^{-\frac{8a}{3}}=0.5$$

Using the natural log again:

$$-\frac{8a}{3}=-\text{ln}(2)$$

Finally solve:

$$a=\frac{3\text{ln}(2)}{8}\approx0.2599$$

What about the probability the insect survives at least three months? We then want to find \( P(T\gt3) \), which is \( G(3) \):

$$G(3)=e^{-\int_{0}^{3}ay^{2}dy}$$

Integration resolves this to \( 9a \), we already calculated the value of the \( a \) constant from before so we can substitute now:

$$9\left( \frac{3\text{ln}(2)}{8} \right)=\frac{27\text{ln}(2)}{8}\approx2.3393$$

Then we can calculate the final result:

$$G(3)=e^{-\frac{27\text{ln}(2)}{8}}\approx0.0963$$

Meaning there is approximately a 9.6% chance that an insect of this kind survives at least three months.
