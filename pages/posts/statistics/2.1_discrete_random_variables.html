2.1 introduces random variables, ranges, discrete random variables and their application in a uniform distribution.

Random Variables

Using random variables in statistics is how we can analyze the outcome of a random experiment with mathematics. A random variable (\( X \)) is defined as a function that maps every possible outcome of a sample space (\( S \)) to a specific real number:

$$X:S\in \mathbb{R}$$

For example, we roll a die twice and sum the results, the sample space is the set of all possible sums:

$$S=\{2,3,\cdots,12\}$$

The function \( X \) (where \( m \) is an outcome within the sample space) assigns a value from this set to each outcome of the experiment; turning the qualitative outcomes into quantitative data:

$$X(m)=m\forall m\in S$$

To clarify, this function doesn't change the number, rather, it just labels the outcome as a numerical value for us to then use in calculations.

This may seem redundant and unnecessary at first, but we will see soon why this is important.

Distributions

The distribution of a random variable is the assignment of probabilities to each value that the random variable can take. For example, we toss a coin twice and count the number of heads we get. Let \( X \) be the number of heads (hence the definition a random variable above):

$$P(X=0)=\frac{1}{4},\quad P(X=1)=\frac{1}{2},\quad P(X=2)=\frac{1}{4}$$

If \( X=0 \) it means we got no heads in either coin flip, and the probability of that is 0.25, the same logic then applies to the other values.

Range

The range of a random variable is the set of all possible values that the random variable \( X \) can be as a result of an experiment. Ranges are categorized based on how many values they contain. This goes back to our introduction of sets/sample spaces in 1.1. But let's review:

Countable Range

As the name implies, these are ranges where the objects can be listed or counted. It can be either:

  • Finite - the range contains a specific number of elements. For example, if \( X \) is the outcome of a single die roll, the range is \( \{1,2,3,4,5,6\} \)
  • Infinitely countable - the range goes on forever, but we can still assign each value a position in a sequence. An example is the number of coin tosses it takes until we get heads, which could be \( \{1,2,3,\cdots\} \) and in theory, could go on forever if you were really that unlucky

Uncountable Range

This can occur when the range consists of an interval that cannot be listed as individual points. For example, if \( X \) represents the lifetime of a proton in seconds, the range would be \( [0,\infty) \) because time is continuous.

Discrete Random Variables

A random variable becomes "discrete" when its range is countable. Meaning we can list every possible outcome (the list is either finite or infinitely countable). We now get to the Probability Mass Function (pmf), which assigns a specific probability to each possible value in the range. If \( X \) is a discrete random variable with the range \( \{x_{1},x_{2},\cdots\} \) then the function is defined as:

$$p(x_{k})=P(X=x_{k})$$

For any function to qualify as a valid pmf for a discrete random variable, its outputs must be greater than or equal to zero, and all possible values in the range must equal exactly one:

$$\sum_{k=1}^{\infty}p(x_{k})=1$$

When we perform an experiment multiple times, the actual observed frequencies are expected to match the distribution as defined by the pmf. Pmf's can also be represented by histograms, which is a great way to see how the probabilities are distributed across the possible outcomes.

Example Pmf Definition

We toss a coin three times and count the number of heads. Our range then looks like:

  • TTT: 0 heads
  • TTH, THT, HTT: 1 head
  • THH, HTH, HHT: 2 heads
  • HHH: 3 heads

$$\text{Range}(X)=\{0,1,2,3\}$$

The pmf would then be:

$$p(0)=\frac{1}{8},\quad p(1)=\frac{3}{8},\quad p(2)=\frac{3}{8},\quad p(3)=\frac{1}{8}$$

Uniform Distributions

A discrete random variable \( X \) has a uniform distribution if every value in its range has the exact same probability. If we have the range \( \{x_{1},x_{2},\cdots\} \), then \( X \) has a uniform distribution if the pmf for each value is defined as:

$$p(x_{k})=\frac{1}{n}\forall k\in\{1,2,\cdots,n\}$$

This applies only to finite ranges. A uniform distribution is impossible if the range is infinite because its not possible to distribute a total probability of one equally across an infinite number of outcomes.
